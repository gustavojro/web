Minhas Platinas do PS4

Nessa página, irei listar os jogos que consegui obter o troféu de Platina, ou seja, jogos no qual peguei todos troféus presentes no game.

Platinar jogos é um hobby que tenho faz algum tempo, mais precisamente desde 2015, onde conquistei meu primeiro troféu de platina no jogo FIFA 15. Desde então peguei o gosto de ir atrás de todos, senão a maioria dos troféus de um game que eu gostei e/ou gosto de jogar. No momento estou no nível 13 e possuo 1010 troféus, sendo 15 deles de platina.

Desbloquear todos os troféus de um game, dependendo do jogo, não é uma tarefa tão simples. Alguns jogos exigem que você conclua 100%, pegue todos os colecionáveis, conclua missões com medalha ouro, elimine certas quantidades de inimigos com determinadas armas enfim, são vários os desafios que temos de enfrentar para conseguir o tão sonhado troféu de platina.
Para algumas pessoas, esses desafios se tornam um bicho de sete cabeças e para outras uma diversão sem tamanho. Mas eu penso que para isso, temos de nos concentrar e ter bastante atenção para poder concluí-los.
